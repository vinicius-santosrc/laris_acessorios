API LARIS ACESSÓRIOS

Diretórios da API

  • /api/v1/{secretKey}/admins - Listar administradores
  • /api/v1/{secretKey}/planilha-despesas - Listar despesas
  • /api/v1/{secretKey}/planilha-itens - Listar itens
  • /api/v1/{secretKey}/metas - Listar metas
  • /api/v1/{secretKey}/planejamentos - Listar planejamentos
  • /api/v1/{secretKey}/products - Listar produtos
  • /api/v1/{secretKey}/orders - Listar pedidos
  • /api/v1/{secretKey}/cupons - Listar cupons
  • /api/v1/{secretKey}/users - Listar usuários

Sobre a API

A API LARIS ACESSÓRIOS oferece uma série de funcionalidades para gerenciar produtos, usuários, pedidos, cupons e muito mais para o sistema de acessórios online.
